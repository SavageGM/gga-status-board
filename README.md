# GGA Status Board

Standalone status board for GURPS Game Aid status commands.

Foundry Manifest URL:
https://raw.githubusercontent.com/SavageGM/gga-status-board/main/module.json

Important: the repository must be public for Foundry to install from this manifest URL. If the repository is private, install the module manually.

Manual install: download the repository ZIP, extract it, and make sure the folder name is exactly gga-status-board. Place that folder in FoundryVTT/Data/modules/. The module.json file must be directly inside the gga-status-board folder.

Use: enable GGA Status Board in Manage Modules. Open from Token Controls, press Alt+Shift+S, or run GGAStatusBoard.open(); from a script macro.

Commands executed:
/status on <status id>
/status off <status id>
/status t <status id>
/status clear
